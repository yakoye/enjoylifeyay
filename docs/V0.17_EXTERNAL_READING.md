# v0.17：阅读站外专题

## 目标

在“阅读”页面保留本站阅读文章，同时补充可长期沉浸的站外长文、独立博客与知识档案。页面保持文本目录风格：名称是链接，后接一句短说明；不做排名、卡片、评分或推荐指数。

## 数据与去重

数据文件：

```text
src/content/reading-sites.json
```

当前收录 96 个站外专题入口，按以下主题分组：

- 工程、系统与程序设计
- 认知、决策与学习
- 科学、未来与社会
- 数字考古与知识档案
- 文学、思想与非虚构
- 生活、独立工作与行走

同一站点即使跨多个主题，也只收录一次，避免页面重复。

## 手动置顶“常看”

将某条目的 `pinned` 改成 `true`：

```json
{
  "id": "paul-graham",
  "name": "Paul Graham",
  "url": "https://paulgraham.com/articles.html",
  "description": "创业、独立思考与创造人们想要的东西。",
  "category": "cognition",
  "pinned": true
}
```

它会自动出现在阅读页“站外专题 → 常看”最前面。`pinned` 只表示你个人常看，不表示网站的客观排名。

## 新增一个站点

复制一个现有对象，保证：

1. `id` 全局唯一，使用小写英文与连字符。
2. `url` 使用完整 URL。
3. `description` 控制为一句中文；优先写出一个记忆锚点或代表性主题。
4. 只选择一个最适合的 `category`。
5. 默认 `pinned: false`。
6. 描述写成一段简短说明即可；桌面端会保持单行，超出当前宽度时用省略号收束并保留悬停全文。名称很长时仍建议相应缩短说明。
7. 不录入不确定、已失效或不希望公开跳转的链接。

修改后运行：

```powershell
.\preview-local.cmd
```
