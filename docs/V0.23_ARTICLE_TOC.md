# v0.23 文章目录

## 桌面端

- 浏览器宽度达到 900px 时，文章使用“左侧目录 + 右侧正文”的双栏布局。
- 目录默认展开，并使用 `position: sticky` 在滚动时保持在可视区域内。
- 点击“收起”后，目录变为带章节圆点的轨道；点击“展开目录”恢复完整目录。
- 完整目录与圆点轨道都可以直接跳转到对应标题。

## 手机端

- 默认只展示前三个标题，避免长目录占满首屏。
- 点击“展开目录”显示全部标题，点击“收起目录”恢复三项预览。
- 目录位于文章信息与正文之间，不遮挡正文。

## 阅读位置

目录会根据页面滚动位置高亮当前章节；完整目录、圆点轨道和手机目录使用同一套章节状态。
