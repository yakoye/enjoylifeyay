# v0.17.2：站外专题扩充与用户描述更新

## 内容

- 已采用用户维护的 `reading-sites.json`，收录 **96** 个站外独立博客、知识档案与长文专题入口。
- 新增的内容主要覆盖：
  - 系统工程、硬件逆向、嵌入式、性能与分布式系统；
  - 半导体产业链、金融基础设施、建设效率与全球科技供应链；
  - Gopher / Gemini、SDF 公共 UNIX 社区等老派互联网与轻量协议空间。
- 分类保持不变：工程、认知、科学、数字考古、文学、生活。
- 每个 `id` 与 `url` 已进行唯一性验证；所有条目均有完整 URL、分类与 `pinned` 字段。

## 呈现

- 桌面端站外专题保持“一行一个网站”。
- 用户提供的较长说明会保留在 JSON；当超过正文宽度时，视觉上使用省略号收束，避免横向溢出。
- 每条 `li` 增加原生 `title`，鼠标悬停可查看完整“名称：说明”。
- 手机端不裁切，允许自然换行。

## 维护

继续编辑：

```text
src/content/reading-sites.json
```

- 常看：将 `pinned` 改为 `true`。
- 新增站点：保证 `id`、`url` 唯一；选择唯一分类；不要重复录入跨领域站点。
- 更新后运行：

```powershell
.\preview-local.cmd
```
