# v0.6：存档与历史内容版本说明

## 面向访客的变化

- `/archive/` 已从“本站已迁入文章列表”升级为完整历史归档。
- 归档按年份显示，日期左侧、标题右侧；样式参考传统博客的年度存档，但不使用红色字体。
- 归档中既包含本站可读正文，也包含 CSDN 和旧博客的历史原文链接。
- 工具页和项目页的公开地址已经补上；未发布工具不显示虚假的链接或待确认状态。
- 浏览器标签页和站点导航使用 Enjoy Life 图标。

## 面向维护者的变化

- 历史来源唯一台账：`docs/CONTENT_MIGRATION_MANIFEST.csv`。
- 归档生成文件：`src/data/legacyArchive.ts`，不手工修改。
- 执行 `npm run sync:legacy-archive` 或 `npm run build` 更新归档。
- 新博客模板：`docs/templates/new-writing.md`。
- 完整操作手册：`docs/CONTENT_MAINTENANCE.md`。
