# v0.9：标题栏、目录与窄屏导航

## 已完成

- `--nav-max` 现在引用 `--content-max`，标题栏内部宽度与正文阅读列保持同宽。
- 顶栏仍使用三栏 Grid：左侧站点、居中主导航、右侧搜索与主题。
- 小于等于 `620px` 时，只隐藏 `Enjoy Life` 的文字标签，不隐藏 favicon 图标；导航保持横向可滚动。
- 文章折叠目录入口显示为“目录”，默认折叠。

## 验收

桌面端检查：顶栏内左、右边界与正文 `890px` 内容列对齐。

手机端检查：在约 390px 宽度下，左侧只保留图标，主页、写作等导航不会被站点名遮挡。
