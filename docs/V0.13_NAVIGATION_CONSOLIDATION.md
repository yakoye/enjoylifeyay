# v0.13：栏目合并与去冗余

## 正式顶栏

```text
主页  技术  工具  阅读  自然  生活  归档  关于
```

站点副标题改为：

```text
技术 · 工具 · 阅读 · 自然 · 生活
```

## 合并规则

| 旧入口 | 现入口 | 说明 |
| --- | --- | --- |
| 写作 | 技术 | 技术文章统一在 `/technology/`；旧 `/writing/` 301 跳转。 |
| 专题 | 技术 | 技术专题统一在 `/technology/#technology-series`；专题详情保留在 `/series/<id>/`。 |
| 工具 | 工具 | 公开工具、个人网站、资料库、长期参考统一在 `/tools/`。 |
| 书架 | 阅读 | 读书文章、书架、阅读专题统一在 `/reading/`。 |
| 自然 | 自然 | 自然观察、植物、地方记忆保留在 `/nature/`。 |
| 生活 | 生活 | 生活、骑行、跑步文章统一在 `/life/`。 |
| 收藏 | 工具 | 收藏作为 `/tools/#references` 的“长期收藏与参考”。 |
| 项目 | 工具 | 项目作为工具页的“个人系统与长期项目”；旧 `/projects/` 301 跳转。 |

## 旧链接

旧入口仍保留重定向，避免书签失效：

```text
/writing/   -> /technology/
/series/    -> /technology/#technology-series
/bookshelf/ -> /reading/#bookshelf
/favorites/ -> /tools/#references
/projects/  -> /tools/#projects
```

## 日常维护位置

| 想新增的内容 | 文件位置 | 发布后出现的位置 |
| --- | --- | --- |
| 技术文章 | `src/content/writing/*.md(x)`，`domain: technology` | 技术 |
| 阅读文章 | `src/content/writing/*.md(x)`，`domain: reading` | 阅读 |

| 书籍 | `src/content/books.json` | 阅读 / 书架 |

| 长期参考、旧站入口 | `src/content/favorites.json` | 工具 / 长期收藏与参考 |
| 专题 | `src/content/series.json` | 技术、阅读或生活的对应页面 |

## `projects.json` 的处理

历史 `src/content/projects.json` 暂时保留为项目规划与旧资料索引，不再作为公开页面的数据源。要让一个新项目出现在网站上，请把它作为工具页条目维护在 `src/content/tools.json`：

- 可直接使用的网页、扩展、资料库：放进对应工具分类；
- 个人站点、长期系统、暂无公开链接的作品：使用 `category: "personal-system"`，显示在“个人系统与长期项目”。
