# v0.4 Text Directory Changes

## What changed

- 顶部导航加入左侧 `Enjoy Life` 品牌链接，并统一为 48px 的文本横栏。
- 导航栏最大宽度与正文分离：导航 1000px，正文 890px。
- 将全站字体、字号、颜色、留白和富文本规则拆分为 `tokens.css` 与 `typography.css`。
- 将工具页、项目页、专题页、书架页、收藏页改为分组文本目录：
  - `· 名称：一句说明。`
  - 有公开地址时，名称本身为链接。
  - 没有公开地址时，名称为普通文字。
  - 不展示“公开地址待确认”、状态徽章和重复分类。
- 写作与自然观察列表使用统一的日期组件：桌面右侧日期、手机标题下日期。
- 项目数据加入 `category` 字段，用于浏览器扩展、个人网站与工具、资料库、私人项目分组。
- 文章页面日期统一为 `YYYY-MM-DD`；普通文章不显示分钟级时间。
- 页脚辅助导航全部使用中文，移除视觉分隔圆点。
- 代码、引用、表格、图片与链接改为无圆角、无阴影、细线和等宽优先的富文本风格。

## Intentionally not changed

- 没有编造任何文章、自然观察、书目或收藏内容。
- 未确认的工具链接、GitHub 链接和私有项目地址仍保持为空，不会输出假链接。
- FamilyJourney、Obsidian 日记系统等私人项目仍然不暴露私有数据。

## Local verification

The package was checked with:

```bash
npm run check
npm test
npm run build
npm run check:links
```

All checks passed before packaging.
