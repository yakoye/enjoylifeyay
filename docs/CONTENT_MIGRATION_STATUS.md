## 迁入规则

- 有完整 Markdown、确认可公开且适合长期保存的旧文：迁入本站正文，保留原始日期和来源。
- 只确认标题、日期、链接的 CSDN 文章：先显示在 `/archive/`，点击回到原平台；不把未核对正文伪装成本站文章。
- 转载、下载、第三方全文、附件或隐私风险文章：保留来源入口，默认不复制正文。
- 知乎：没有确认的标题、日期和正文时不猜测；等待账号导出或站长提供文章链接后再迁入。

## v0.7 图片迁移状态

已迁入 Markdown 的旧 EnjoyLifeBlog 图片链接目前仍可作为原始来源保留。项目新增 `npm run import:legacy-images -- --dry-run` 与 `--write`：站长在可访问旧站的本地网络环境下执行后，图片会复制到 `public/images/legacy/`，并在下一次 Cloudflare Pages 发布时改由本站提供。该步骤没有在打包环境中自动执行，避免无法验证的远程下载造成缺图或静默替换。
