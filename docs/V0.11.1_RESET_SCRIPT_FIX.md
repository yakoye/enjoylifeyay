# v0.11.1：Windows 本地清理脚本修复

## 修复内容

此前 `scripts/reset-local.ps1 -StopAllNode` 直接调用：

```powershell
taskkill /F /T /IM node.exe
```

当电脑上没有任何 Node 进程时，Windows 会输出 `The process "node.exe" not found.`。虽然这不代表项目有问题，但输出看起来像报错。

现在脚本先用 `Get-Process -Name node -ErrorAction SilentlyContinue` 检查。

- 有 Node 进程：逐个强制结束，再继续清理。
- 没有 Node 进程：提示 `No node.exe processes are running; continuing.`，继续清理。

## 正常用法

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\reset-local.ps1 -StopAllNode
npm ci
```

不要因为“没有 node.exe 进程”而重复运行或中止后续 `npm ci`。
