# v0.17.3：站外专题扩充数据与两行说明

## 内容

- 已采用用户最新维护的 `reading-sites.json`，共 **96** 个站外独立博客、知识档案与长文专题入口。
- 新增和扩充的工程类内容包括硬件逆向、嵌入式、系统性能、分布式系统、安全思想与老硬件档案。

- 分类维持六组：工程、认知、科学、数字考古、文学、生活。

## 呈现

- 名称本身是外部链接，在新标签页打开。
- 无排名、编号或评分。
- `pinned: true` 会进入“常看”分组，不代表推荐等级。
- 桌面端说明允许自然使用 **最多两行**；超过两行时收束，不会撑出横向滚动或卡片化布局。
- 手机端不进行行数裁切，允许自然换行。
- 每项保留原生 `title`，鼠标悬停可查看完整的“名称：说明”。

## 轻微编辑

为避免明显重复或句法问题，保留用户文字的基础上仅处理了三处：

- Conceptually：补上“把”。
- Kevin Kelly / The Technium：删除重复的 `Technium`。
- Textfiles：移除描述中的重复入口 URL，名称本身已是可点击链接。

## 维护

继续编辑：

```text
src/content/reading-sites.json
```

每条需要：唯一 `id`、唯一 `url`、一句 `description`、分类 `category` 和布尔 `pinned`。

更新后运行：

```powershell
.\preview-local.cmd
```
