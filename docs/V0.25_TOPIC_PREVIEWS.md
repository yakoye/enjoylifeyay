# v0.25 一级栏目专题预览

## 页面结构

五个一级栏目页统一采用下面的内容顺序：

```text
一级栏目标题
栏目简介

二级主题 A
- 标题                                      日期
  概要
  一级栏目 · 二级主题 · 标签
- 最多三条
查看更多 >>

二级主题 B
……
```

## 展示规则

- 每个二级主题最多预览三条内容。
- 文章和自然观察按日期倒序；工具按维护顺序；书架按最近阅读时间排序。
- 同一内容在同一个一级栏目页只出现一次。
- 没有独立内容的空专题不重复占位；纯 Markdown 专题仍保留入口和简介。
- 技术页的“技术文章”承担综合入口，内容量已经形成专题的 PCIe 等栏目保留自己的最近三条。

## 列表排版

文章和带日期的观察记录统一显示为：

```text
标题                                      日期
概要
一级栏目 · 二级主题 · 标签
```

桌面和手机使用相同的信息顺序；手机端长标题可以换行，日期仍位于标题行右侧。
