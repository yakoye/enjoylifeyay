# v0.11：评论管理、写作列表与 MSI / MSI-X 文章

## 完成内容

- 移除写作页“领域 / 形式”筛选：当前内容量不大，筛选无法稳定工作时不应占用页面。
- 评论区改为两列单行昵称 / 邮箱与一行自适应留言；三个字段均使用极简线性 SVG 图标。
- 邮箱为可选字段：仅存 D1，不公开显示。
- 新增 `COMMENTS_MODERATION=auto` 可选自动公开模式；默认仍为 `pending` 审核。
- 新增远程 D1 命令行管理：查看、通过、拒绝、删除、状态、API 探测、邮箱列迁移。
- 新增旧 旧个人站 文章《PCIe MSI / MSI-X：Capability、Table 与 Linux 驱动接口》，包含本地图片、表格、代码块、引用和目录，用于验证技术长文渲染。

## 发布顺序

```powershell
npm ci
npm run check
npm test
npm run audit:toc
npm run build
npm run check:links
npm run comments:migrate
npx wrangler pages deploy dist --project-name enjoylifeyay --branch main
npm run comments:probe -- pcie-msi-msix-introduction
```

如果先前已绑定 D1，但 `/api/comments` 返回 `configured:false`，重新部署后再运行 `comments:probe`。
