# Enjoy Life 基础骨架 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 建立可由 Cloudflare Pages 部署的 Astro 静态个人站，完成内容模型、全局阅读系统、主题、导航、首页、写作列表和文章详情页。

**Architecture:** 采用 Astro 静态输出与 Content Collections，内容保存在仓库内的 Markdown/MDX；页面在构建期读取并过滤草稿。客户端仅保留主题偏好脚本，不引入 React、SSR、数据库或 CMS。

**Tech Stack:** Astro 5、TypeScript、Markdown/MDX、原生 CSS、Node.js 22、npm、Node test runner
