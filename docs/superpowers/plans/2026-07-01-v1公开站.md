# Enjoy Life v1 公开站 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将已发布的 Astro 骨架完善为具备真实内容目录、RSS、Sitemap、静态搜索、SEO、迁移台账与可验证路由的长期维护版 v1。

**Architecture:** 保持 Astro 静态输出和现有极简视觉，将公开资料存入七类 Content Collections，所有页面仅读取 `draft: false` 条目。站点 URL、导航和 SEO 信息集中在 `src/config/site.ts`，Pagefind 在构建后对 `dist/` 建立本地索引。

**Tech Stack:** Astro 7、TypeScript、Markdown/JSON Content Collections、原生 CSS/JavaScript、Pagefind、`@astrojs/rss`、`@astrojs/sitemap`、Node test runner
