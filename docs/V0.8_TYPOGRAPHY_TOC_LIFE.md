# v0.8：标题节奏、目录、书架日期与生活运动专题

## 已完成

- 页面一级标题收敛为 26px；首页 `Enjoy Life` 与栏目页标题保持同一字号。
- 栏目页标题与简介之间调整为约 3px；首页标题与副标题继续保持零额外间距。
- 首页模块间隔、专题分组间隔、写作筛选区间隔收紧。
- 内容区域所有 `hr` 与 `.rule` 隐藏；只保留顶部导航和页脚的功能性细线。
- `updated: null` 在内容 schema 中先匹配 `null`，不会再被 `z.coerce.date()` 转为 Unix epoch。文章布局也会过滤 epoch 日期，避免继续显示 `最后修订：1970-01-01`。
- 长文章在标题信息后自动输出默认折叠的 `[ToC]`。
- 书架使用带右侧日期的目录列表：已读书目显示 `finishedAt`，在读书目显示 `startedAt`。
- 新增 `life-cycling` 与 `life-running` 专题，显示在 `/series/` 的“生活与运动”分组。
- 写作筛选由可靠的打包模块脚本执行；点击筛选、URL 同步、浏览器前进后退均会重新渲染条目。

## 后续内容维护

- 骑行文章：`domain: life` + `series: ["life-cycling"]`。
- 跑步文章：`domain: life` + `series: ["life-running"]`。
- 没有实际修订日期时：`updated: null`。
- 不要手动插入 Markdown 风格的横线作为页面装饰。
